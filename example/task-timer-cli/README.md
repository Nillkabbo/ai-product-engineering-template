# Task Timer CLI

A lightweight CLI tool for developers to track time spent on coding tasks.

## Overview
This project demonstrates the AI Product Engineering template in action. It's a simple Python tool that lets you start/stop timers, view time logs, and export to CSV.

## Installation
```bash
python timer.py --help
```

## Usage
```bash
# Start a timer
python timer.py start "Working on feature X"

# Stop the active timer
python timer.py stop

# View all time entries
python timer.py list

# Export to CSV
python timer.py export
```
