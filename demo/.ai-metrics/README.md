# AI Metrics

Purpose
Capture basic model usage and cost notes per AI session.

Rules
- Record one entry per task
- Keep entries short and factual
- Link to the task file

How to Use
1. Copy session-template.md into a new file in .ai-metrics/
2. Name the file YYYY-MM-DD-task-name.md
3. Fill in model, tokens, cost estimate, and outcome
