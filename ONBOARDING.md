# 1-Page Coworker Onboarding

## Quick Start
1. Read MASTER_REFERENCE.md once
2. Pick a task in tasks/
3. Use SYSTEM_PROMPT.md in your AI tool
4. Update WORKING_STATE.md when done

## File Map
- ARCHITECTURE.md: Law. Must not be violated.
- tasks/: Your work queue and task record
- WORKING_STATE.md: Team status and decisions

## AI Usage Rules
- No task file -> no AI work
- One task at a time
- Minimal diff only
- Update WORKING_STATE.md before ending

## First Task Walkthrough
1. Copy tasks/TASK_TEMPLATE.md
2. Fill in Goal, Context, Deliverables, Acceptance Criteria
3. Record model selection
4. Complete task and update WORKING_STATE.md
